# 🚀 MediaShuttle Bot

Попробовать бота в Telegram: @media_shuttlebot                                         
Асинхронный Telegram-бот на **Aiogram 3**, который не просто качает видео, а понимает их.

## ✨ Основные функции
- **🎬 Скачивание:** Видео (MP4) и Аудио (MP3).
- **🧠 AI Анализ:** Мультимодальный пересказ видео через **Gemini 2.5 Flash** (нейронка "смотрит" видеоряд).
- **🐳 DevOps:** Полный Docker-пакет с автоматической установкой FFmpeg и Node.js.
- **🔐 Обход блокировок:** Реализована поддержка Cookies и JS-рантайма для стабильной работы с YouTube.

## 🛠 Стек технологий
- **Backend:** Python 3.10+, Aiogram 3.4
- **AI:** Google GenAI SDK
- **DB:** PostgreSQL + asyncpg
- **Infrastructure:** Docker, Docker Compose
